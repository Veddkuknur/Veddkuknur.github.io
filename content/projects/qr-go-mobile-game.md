## Project Overview

QR-GO is an Android mobile game where players collect QR codes at various locations. Built as a team project for CMPUT 301, this game combines location-based gameplay with social features, demonstrating expertise in Android development, Google Maps integration, and Firebase backend services.

## Key Features

### QR Code Collection
- **Location-Based Gameplay**: Scan QR codes at different locations
- **Collection System**: Track and manage collected QR codes
- **Scoring System**: Points based on QR code rarity and location

### Google Maps Integration
- **Interactive Maps**: Google Maps SDK for location visualization
- **Location Tracking**: Real-time location services
- **Marker Management**: Display QR code locations on map

### Firebase Backend
- **Cloud Database**: Firebase Realtime Database for data storage
- **Authentication**: User authentication and profiles
- **Social Features**: Leaderboards and user interactions

## Performance Improvements

### 60% Crash Reduction
- **Threading Optimization**: Proper background thread management
- **Async Operations**: Asynchronous data loading and processing
- **Error Handling**: Robust error handling and recovery mechanisms
- **Memory Management**: Efficient memory usage patterns
