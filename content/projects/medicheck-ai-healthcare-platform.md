## Project Overview

MediCheck is an AI-powered healthcare platform built during the HackED hackathon that secured **3rd place**. The platform combines cutting-edge AI technologies to provide intelligent medical assistance through a voice-enabled chatbot interface.

## Key Features

### Voice-Enabled Chatbot
- **Speech-to-Text**: Integrated OpenAI Whisper for accurate voice transcription
- **AI Assistant**: Powered by DeepSeek and Llama 3.2 for intelligent responses
- **Natural Language Processing**: Conversational interface for medical queries

### RAG Pipeline
- **Vector Database**: AstraDB integration for efficient knowledge retrieval
- **Context-Aware Responses**: Retrieval-Augmented Generation for accurate medical information
- **Real-time Processing**: Fast response times for user queries

### Modern Tech Stack
- **Frontend**: Next.js with React and TypeScript
- **Styling**: Tailwind CSS for responsive design
- **Backend**: Python for AI/ML processing
- **Database**: AstraDB for vector storage and retrieval
